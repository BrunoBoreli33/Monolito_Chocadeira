# APLICAÇÃO DE SISTEMAS EMBARCADOS NO CONTROLE DE INCUBADORAS: INTERFACE INTERATIVA E CONFIGURAÇÃO FLEXÍVEL DE PARÂMETROS AMBIENTAIS E MECÂNICOS

Esse projeto teve como objetivo elaborar um protótipo de chocadeira automatizada, com interface interativa para ajuste de parâmetros ambientais e mecânicos, visando aprimorar a qualidade do processo e simular condições ideais para a incubação de ovos.
- O trabalho foi meu TCC na graduação do curso de Engenharia da Computação na UNIFEITEP.
- Obteve nota máxima de todos os professores da banca.
---

## Objetivos do Projeto

- Implementar um sistema PID para controle de temperatura
- Desenvolver interface interativa para controle de parâmetros ambientais e mecânicos
- Contorle on/off para umidade
- Plataforma de ovos com inclinaçao automática e totalmente programável

## Após a construção

Passos para a execução do código:

- Instalar o Arduino IDE
- Instalar todas as bibliotecas .zip do repositório na IDE do arduíno
- Compilar o código "Main.c" e mandar o código compilado para o arduino.

## Resultado Final

- O controle de temperatura PID teve apenas 1% de variação após estabilizar.
- Interface completa para configuração, controle e monitoramento da chocadeira.
- Controle de umidade feito por relés on/off.
- Mecanismo de Viragem de ovos automático e totalmente programável.
